## 🧠 核心技术

### 受 Queryable 启发
```
Queryable: 文本 → CLIP → 理解图像
本插件: 文本 → Sentence-BERT → 理解文本
```

### 工作原理
```
1. 每个书签 → 384维语义向量
2. 你的查询 → 384维语义向量
3. 计算相似度 → 返回最相关的

就像 Queryable 用 CLIP 理解图像，
我们用 Sentence-BERT 理解文本！
```
